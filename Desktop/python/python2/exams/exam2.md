10. Дар куҷо метавонистед closure, args ва kwargs-ро якҷоя истифода баред?



1. Recursion

Савол: Ба воситаи рекурсия функсияе нависед, ки ҷамъи ҳамаи элементҳои рӯйхатро ҳисоб кунад.

def fun(a):
    if a==1:
        return 1
    return a+fun(a-1)
a=int(input())
print(fun(a))

2. Recursion

Савол: Як функсия бо рекурсия нависед, ки ҳарфи аввал ва охирро дар як сатр мубаддал кунад.

3. Closure

Савол: Як closure эҷод кунед, ки барои ҳар як рақам дар рӯйхат онро ба қувваи муайян боло бардорад.

def outter(lst):
    ls=[]
    def inner(x):
        nonlocal ls
        for i in lst:
            ls.append(i**x)
        return ls
    return inner

a=outter([9,2,3])
print(a(3))

4. String

Савол: Барои сатре, ки ворид мешавад, ҳарфи калон ва хурдро ҳисоб кунед.

a=input()
sm=0
bg=0
for i in a:
    if i.islower():
        sm+=1
    elif i.isupper():
        bg+=1
print(sm,bg)


5. String

Савол: Функсияе нависед, ки баръакс кардани сатр (reverse) ва шумоҳадиси он нишон диҳад.

a=input()
print(a[::-1])


6. Function + args

Савол: Функсияе эҷод кунед, ки ҳар чанд адад воридшударо ҷамъ кунад, ва шуморо иҷозат диҳад, ки ададҳо ҳар қадаре ки мехоҳед ворид шаванд.

def fun(*args):
    sm=0
    for i in args:
        sm+=i
    print(sm)
a=[2,3,4,11,5]
fun(*a)

7. Function + kwargs

Савол: Функсия бо kwargs эҷод кунед, ки маълумоти шахсро (ном, насаб, синну сол, шаҳр) пешкаш кунад ва чоп кунад.

def fun(**kwargs):
    for k,v in kwargs.items():
        print(f'{k} : {v}')
    
dct={
    'name':input(),
    'surname':input(),
    'age':int(input()),
    'city':input()
}
fun(**dct)

8. Set

Савол: Як рӯйхатро ба set табдил диҳед ва ҳама элементҳои такроршавандаро хориҷ кунед.

a=[1,2,3,4,4,5]
a=set(a)
print(a)

9. Try-Except

Савол: Функсияе нависед, ки аддиро аз корбар бипурсад ва агар адад хато бошад, хатоеро идора кунад.

try:
    a=int(input())
    print(a)
except ValueError:
    print('Value is error!')

10. Combinations

Савол: Як функсия нависед, ки рӯйхати ададро қабул мекунад, ададҳои манфиро хориҷ мекунад, ва дар ниҳоят ҷамъи боқимондаро ҳисоб кунад, бо истифода аз рекурсия, set, ва try-except.
